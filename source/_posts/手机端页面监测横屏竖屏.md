在做手机端网页时，如果没有做横竖屏自适应，那么横竖屏监测是个很好的用户体验@#￥%……%*&……%代码：
```
var updateOrientation = function(){
	if(window.orientation == '-90' || window.orientation == '90'){
		alert("请将手机竖屏");
	}else{
		 alert("现在是竖屏");
	}
}
window.onorientationchange = updateOrientation;
```
